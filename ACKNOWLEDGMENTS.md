This project is a **Teneo Protocol Agent** that leverages the powerful agent framework to provide on-chain intelligence services.

### Key Dependencies

#### [Teneo Agent SDK](https://github.com/TeneoProtocolAI/teneo-agent-sdk)
The core framework powering this agent:

- Agent lifecycle management
- Multi-chain support integration

#### [go-ethereum](https://github.com/ethereum/go-ethereum)
Blockchain interaction:
- RPC client for chain queries
- Smart contract code retrieval
- Balance checking across chains

#### [CoinGecko API](https://www.coingecko.com/api)
Real-time market data:
- Cryptocurrency pricing
- Multi-currency support
